To compile use make
To run use ./a.out

In app:
    w,s,up,down,mouse -> move the robot
    mouse right click -> reset camera
    mouse wheel -> zoom
    z -> Camera Superior
    ESC -> exit
    5 -> Camera Robo
    1 -> Camera da Janela 
    2 -> Camera do android 
    3 -> Camera da Chá
    4 -> Camera do Torus 
    6 -> Camera Baude Nurbs
    7 -> Camera do Troféu Nurbs
